## 简介
    npm install
### 运行
    npm run dev
### 编译
    npm run build
访问：http://localhost:8080/
### 制作自己的简历
clone项目，进入master分支。在static目录下找到resume.json文件，按照自己的情况修改相应的信息即可。

    ctrl+p
